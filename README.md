# Financial-Analysis & Reporting-Dashboard

## 📌 Overview

This project is an interactive Financial Dashboard built using Microsoft Excel to analyze and visualize key financial metrics such as revenue, expenses, profit, and overall financial performance.

It provides a clear and structured view of business financial data using Excel features like **Pivot Tables, Charts, Slicers, and Conditional Formatting**.

---

## 🎯 Objective

* To analyze financial performance using Excel
* To track revenue, expenses, and profit trends
* To create an interactive dashboard for decision-making
* To simplify financial reporting through visualization

---

## 🧰 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers & Filters
* Conditional Formatting
* Formulas & Functions

---

## 📂 Dataset Information

The dataset contains financial data such as:

* Revenue
* Expenses
* Profit
* Time period (monthly/quarterly)
* Category-wise breakdown 

---

## 📈 Key Features

* 📊 Revenue vs Expense comparison
* 📉 Profit trend analysis over time
* 📅 Time-based filtering (Month/Year)
* 🧾 Category-wise financial breakdown
* 🎛️ Interactive slicers for dynamic filtering
* 📌 KPI indicators for quick insights

---

## 🔍 Key Insights

* Identification of profit and loss trends
* Understanding expense distribution
* Monitoring financial performance over time
* Supporting better financial decision-making
